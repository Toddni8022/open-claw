# Contributing to OpenClaw

We welcome contributions of all kinds — bug reports, feature suggestions, code improvements, and documentation.

## Quick Start

1. Fork & clone the repo
2. `npm install`
3. Create a feature branch: `git checkout -b feature/my-change`
4. Make your changes and run `npm run lint`
5. Open a pull request against `main`

## Full Guide

See [docs/contributing.md](docs/contributing.md) for detailed contribution guidelines including:

## Code of Conduct

Be kind and respectful. This is a collaborative, creative project — everyone is welcome.
